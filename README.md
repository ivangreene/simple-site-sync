# Simple Site Sync
Example of how to synchronize simple sites to an rsync-accepting destination
(i.e, with scp/sftp/ftp/ssh access)

## Setup
- Copy `./.env-example` to `.env` and fill it in.
- Put any tracked files to exclude in `excluded`. See [Excluding files](#excluding-files)
- Put your content in files here (e.g, index.html)
- Or, use `make pull` to pull from the remote

## Usage
- `make push` to push content as it changes

### Excluding files
A dependency of the push target is building the ignored .rsync-exclude.
Do not edit this file directly.
Instead, put any git-tracked files you want to exclude from synchronization
into `./excluded`.
Files from `./.gitignore` will be combined with this list to generate
an rsync exclusion file.
